# HOW TO USE
1. Install 2 extenstion from vscode market place: 
    * Community Server Connectors.
    * Extension Pack for Java.

2. Install tomcat 8.5 and maven in your local machine.

3. [Setup tomcat server in vs code](https://stackoverflow.com/questions/76013263/how-can-we-create-a-dynamic-web-project-in-vs-code)

4. Ctrl + Shift + p then choose "Task: run task", click on "Maven: clean package".

5. target folder will appear, open it, right click on template.war, Run on server.

6. Under the server tab, right click on the server you created, then choose the option "server action", choose "show in browser".

7. Done.
